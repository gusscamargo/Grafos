# Grafos

Como default a classe toma como base os dados dentro do arquivo "arquivo.txt" mas é possivel mudar isto fornecendo o caminho/nome + extensão do arquivo durante a declaração da classe.

O modelo de dados segue o seguinte padrão:

vertice1: vertice2@valorAresta1
vertice2: vertice3@valorArersta2

Algoritmos presentes em Grafos.py dentro da classe Grafo:

- BFS
- DFS
- Fleury
- Dijkstra
- BellmanFord
